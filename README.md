# Commit Message Conventions

## Format of the commit message

<type>(<scope>): <subject>

fix(header): remove overflow-hidden
style(h1): add font-weight 900 to .titre-bleu

### Types

feat       (feature)
fix        (bug fix)
docs       (documentation)
style      (formatting, missing semi colons, etc.)
refactor
test       (when adding missing tests)
chore      (maintain)
